# Customer Churn Analysis Using Machine Learning

## Project Overview

Customer churn is one of the most important challenges for subscription-based businesses. This project predicts whether a customer is likely to leave the service using machine learning techniques. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and business recommendations.

---

## Objective

Develop a classification model to predict customer churn and identify the key factors influencing customer retention.

## Key Insights

- Month-to-month customers have the highest churn.
- Customers with shorter tenure are more likely to leave.
- Higher monthly charges increase churn probability.
- Contract type significantly impacts customer retention.

---

## Business Recommendations

- Promote long-term contracts.
- Offer loyalty rewards.
- Reduce churn through personalized retention campaigns.
- Identify high-risk customers using predictive analytics.
